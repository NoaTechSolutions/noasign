## Modelo público

### Pay-per-contract — $12/doc

Sin suscripción. Para 1–2 documentos al mes.

Incluye:
- crear y enviar documento
- PDF firmado + audit trail
- historial 90 días

No incluye: dashboard operativo, templates guardados, equipo.

## Reglas internas

- **Plan target de ventas:** Launch
- No descontar mensualidad + setup + overage al mismo tiempo
- Founder pricing: máximo $29/mes, cap 10 docs, 1 template, sin setup incluido
- No vender roadmap como features actuales
- El canal B2B API es exclusivamente interno (NTSolutions). Nunca mencionarlo en ventas ni marketing al público.

---

## Regla de upsell

```
1–2 docs/mes          → Pay-per-contract $12/doc
3+ docs/mes           → Starter $19/mes (mejor valor)
Equipo + user mgmt    → Launch $39/mes ⭐
Alto volumen + branding → Pro $89/mes
150+ docs + soporte   → Scale $229/mes
```
